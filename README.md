# Linked-List-ADT
Implementation of a doubly linked list ADT in Java

The list stores elements using nodes that point to the next and previous node.  
It supports common operations like adding, removing, searching, and getting elements.

## Files
- LinkedListADT.java – implementation of the linked list
- ListADT.java – interface for the list
- Main.java – test program for the list methods

## Features
- Add elements to the list
- Insert elements at a specific index
- Remove elements
- Search for elements
- Get element by index
- Find first and last occurrence
- Check the size of the list
